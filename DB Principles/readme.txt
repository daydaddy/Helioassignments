What is a Database?

Answer: A database is an organized collection of structured information, or data,
typically stored electronically in a computer system.

Why is a Database necessary? 

Answer: A database stores and manages a large amount of data on a daily basis. 
This would not be possible using any other tool such as a spreadsheet as they would simply not work. 

What can be stored in a Database? 

The purpose of every database is to store information, texts, images, even media files.

What types of Databases are there? 

Answer: text database, desktop database programs, elational database management systems (RDMS),
NoSQL and object-oriented databases, graph database.

Why would I use one over the other?

What is a Query? 

Answer: A query is a way of requesting information from the database. A database query 
can be either a select query or an action query. 

What is Encryption?

Answer: Encryption is the process that scrambles readable text so it can only be read by 
the person who has the secret code, or decryption key.

Why is it important at the Database level? 

Answer: It keeps sensitive information secure.

What is Object Permanence? 

Answer: Object permanence means knowing that an object still exists, even if it is hidden.
